Node.js Express API

Welcome to your Node.js Express API! This is a simple frontend view to test your API endpoints.

Available Endpoints:

GET /api
Get API information
GET /api/projects
Get all projects
GET /api/projects/:id
Get project by ID
POST /api/projects
Create new project
PUT /api/projects/:id
Update project
DELETE /api/projects/:id
Delete project
GET /api/food
Get all food items
GET /api/hello
Simple hello message
GET /health
Health check endpoint

Testing the API:

You can test these endpoints using tools like:

  • Postman
  • cURL
  • Browser (for GET requests)
  • JavaScript fetch API
